Dicionário de Tags HTML

Este dicionário contém as principais tags utilizadas no desenvolvimento das páginas index.html e html_dict.html

<!DOCTYPE html>

Define o tipo de documento como HTML5.

<html>

Tag raiz que envolve todo o conteúdo HTML da página. O atributo "lang" especifica o idioma da página.

<head>

Contém metadados e informações sobre o documento que não são exibidos diretamente na página.

<meta>

Define metadados como codificação de caracteres e configurações de viewport para responsividade.

<link>

Usado para vincular recursos externos como arquivos CSS e favicons.

<title>

Define o título da página que aparece na aba do navegador.

<body>

Contém todo o conteúdo visível da página web.

<header>

Define o cabeçalho da página, geralmente contendo logo, título e navegação.

<nav>

Define uma seção de navegação com links para outras páginas ou seções da página atual.

<div>

Elemento de divisão genérico usado para agrupar conteúdo e aplicar estilos.

<img>

Insere uma imagem na página. Atributos como "src", "alt", "width" e "height" são usados para configurar a imagem.

<a>

Cria links para outras páginas, arquivos ou locais dentro da mesma página.

<h1>, <h2>, <h3>

Definem cabeçalhos de diferentes níveis de importância, sendo h1 o mais importante e h6 o menos importante.

<p>

Define um parágrafo de texto.

<aside>

Define conteúdo relacionado mas separado do conteúdo principal, como uma barra lateral.

<main>

Define o conteúdo principal da página, excluindo cabeçalhos, rodapés e barras laterais.

<section>

Define uma seção temática do documento, agrupando conteúdo relacionado.

<ul>

Define uma lista não ordenada.

<li>

Define um item de lista dentro de elementos ul ou ol.

<footer>

Define o rodapé da página, geralmente contendo informações de contato, direitos autorais e links adicionais.

<address>

Define informações de contato para o autor/proprietário da página.
